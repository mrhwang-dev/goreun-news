# 뉴스 소스 저작권 검토 (2026-07-22)

광고 기반 무료 AI 뉴스 요약 사이트의 원소스를 조사한 결과 요약.
결론: **"공공 콘텐츠는 본문 AI 요약 + 언론사 뉴스는 제목·링크 큐레이션"** 이원 구조.

## 사용하는 소스

### 1. 대한민국 정책브리핑 (korea.kr) — 본문 AI 요약 허용

- 정책뉴스는 **공공누리 제1유형**: 출처표시만 하면 상업적 이용·변형(AI 요약 포함) 허용
- robots.txt 전체 수집 허용 (`User-Agent: * / Allow: /`)
- 주의: 기사 내 사진·이미지는 제3자 저작물 → **텍스트만 사용**
- 출처표시 의무: "출처: 대한민국 정책브리핑(korea.kr)" (사이트 푸터·섹션에 표기)
- 참고: korea.kr 자체 RSS 서비스는 중단됨 → 목록 페이지에서 직접 수집

### 2. 언론사 공개 RSS — '제목 + 원문 링크'만

법적 근거:
- **제목**: 짧은 기사 제목은 창작성 결여로 저작물성이 부정됨 (판례·한국저작권위원회 실무)
- **링크**: 대법원 2009다4343 — 인터넷 링크는 복제·전송·전시 어디에도 해당하지 않음
- **본문·발췌는 사용 금지**: 창작적 표현의 복제는 분량이 짧아도 침해 소지.
  영리 서비스는 저작권법 제28조 '인용' 항변도 인정되기 어려움

사용 피드 (2026-07-22 응답 확인):

| 언론사 | 피드 | 비고 |
|---|---|---|
| 조선일보 | chosun.com/arc/outboundfeeds/rss/ | 제공 중 |
| 경향신문 | khan.co.kr/rss/rssdata/total_news.xml | 제공 중 |
| 동아일보 | rss.donga.com/total.xml | 제공 중 |
| 한국경제 | hankyung.com/feed/all-news | 제목만 제공 |
| 세계일보 | segye.com/Articles/RSSList/segye_recent.xml | 제공 중 |

제외한 피드와 사유:
- **SBS**: 피드에 "개인 이용자의 비상업적 사용에만 허가" 명시 → 상업 이용 불가
- **한겨레**: 피드에 "무단 전재, 재배포, AI 학습 및 활용 금지" 명시
- **매일경제**: 봇 차단(403)
- **중앙일보**: RSS 중단, **연합뉴스**: 공개 RSS 미제공 (콘텐츠 유료 판매 모델)

### 3. 커뮤니티 인기글 — '게시글 제목 + 원문 링크'만

뉴스 헤드라인과 동일한 법적 근거(제목 저작물성 부정 + 링크 비침해)를 적용한다.
본문·이미지 수집 금지, 제목 원문 그대로 + 아웃링크.

| 소스 | 수집 방식 | 확인 사항 (2026-07-22) |
|---|---|---|
| 루리웹 베스트 | HTML (bbs.ruliweb.com/best/all/now) | robots.txt에서 /best 허용 |
| 더쿠 HOT | HTML (theqoo.net/hot) | robots.txt 미제공, 공지글 필터 적용 |

제외: 클리앙(추천 페이지 410), 오늘의유머(RSS 중단), 개드립(403 차단),
에펨코리아(수집 차단 정책), 뽐뿌(핫게시물 파싱 불가).

## 사용하지 않기로 한 소스

- **네이버 뉴스 검색 API**: 이용약관 금지행위에 ① API 결과화면과 광고 병행 노출 금지,
  ② 취득 정보의 무단 복제·저장·가공·배포 금지, ③ 검색결과 독립 노출·수정 금지가 명시.
  광고 기반 AI 요약 사이트와 정면 충돌 → 사용 불가
- **빅카인즈(BIG KINDS)**: 무료 범위(본문 200자)도 전재·복제·배포 금지, Open API 유료 전환
- **언론사 본문**: 크롤링 + AI 요약은 복제권 침해 소지 + 언론계가 현재 가장 적극
  대응하는 영역 (2025년 한국신문협회의 네이버 공정위 제소 추진 등).
  본문 기반 서비스를 원하면 뉴스토어(newstore.or.kr) 유료 이용권 또는 언론사 개별 제휴 필요

## 운영 수칙

1. 언론사 기사 본문·발췌를 수집·저장·표시하지 않는다
2. 헤드라인은 원문 그대로(수정 없이) 표시하고 반드시 원문에 아웃링크한다
3. 이슈 요약문은 헤드라인에 담긴 정보만으로 AI가 새로 작성한다
4. 정책 브리핑에는 공공누리 출처표시를 유지한다
5. 피드별 이용 조건 문구가 바뀌는지 주기적으로 확인한다 (비상업 한정·AI 금지 문구 추가 추세)
